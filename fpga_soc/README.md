Harsh Environment CubeSat Payload SoC solution to be embedded in the MS2010_VF400 SmartFusion2 Microsemi FPGA.

Libero Release v11.8 (Version 11.8.0.26)

ModelSim 10.5c

Preliminary Design:

ARM Cortex-M3 Core

SDR SDRAM Controller

Watchdog Timer

Timer

UART, SPI, I2C and GPIOs
